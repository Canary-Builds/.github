# Canary Builds

**Open-source tools for resilient infrastructure.**

We build self-healing systems, monitoring tools, and automation software that keep your services running when things go wrong.

| Project | Description | Install |

| [**OpenClaw Aegis**](https://github.com/Canary-Builds/openclaw-aegis) | Self-healing sidecar for the OpenClaw Gateway. 10 health probes, automatic recovery, out-of-band alerts. | `npm install -g openclaw-aegis` |

---

### What We Care About

- **Resilience over uptime** — things will break, so build systems that fix themselves
- **Out-of-band by default** — alerts should never depend on the thing that's broken
- **Zero-config where possible** — auto-detect, sensible defaults, get out of the way
